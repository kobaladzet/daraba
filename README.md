# Daraba

პროექტის დახმარებით, შეგიძლიათ წამოიღოთ daraba.art-ის ოფიციალური ვებ-გვერდის პირველ 5 გვერდზე ატვირტული ნამუშევრების სათაურები, ავტორები და მათი ფასები, რომლებიც შეინახება csv ფაილში.

•	requests

•	Time

•	random

•	BeautifulSoup

•	Csv
